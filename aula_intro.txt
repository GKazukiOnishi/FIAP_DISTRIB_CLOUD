Conceitos Gerais
	Definição de Sistema Distribuído:
		Aquele que se apresenta como sistema centralizado mas funciona em diversas CPUs independentes
		Estabilidade - Se algum componente quebra o sistema todo não cai
		Para nós só estamos usando o sistema, mas não sabemos onde está
	
		Computadores conectados em rede que se comunicam e coordenam suas ações
			- Os componentes não precisam estar próximos entre si, não há limite
			- Para os usuários é uma entidade única, mesmo que funcionem em computadores diferentes de locais diferentes
			
		Ou seja: Conjunto de computadores independentes que se apresenta a seus usuários como um sistema único e coerente
